 # Title & Project name
  🎵 Music Player Web App
 # Project Description
A simple and responsive music player built with HTML, CSS, and JavaScript. 
This project was created as part of an internship task to demonstrate audio control and UI interaction.
# Features
## Features
- Play, pause, and navigate through songs
- Volume control
- Playlist display and selection
- Responsive design
# Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
